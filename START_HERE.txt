🚀 QUICK START (3 Steps):

1. GET RESEND API KEY (2 minutes)
   → Visit: https://resend.com/signup
   → Sign up (free, no credit card)
   → Go to API Keys → Create new key
   → Copy the key

2. ADD TO .ENV FILE
   → Open: project/.env
   → Replace: RESEND_API_KEY=your_resend_api_key_here
   → With your actual key

3. START THE APPLICATION

   OPTION A - Easy (Windows):
   → Double-click: project/start.bat

   OPTION B - Manual (2 terminals):
   
   Terminal 1:
   cd project
   npm run server
   
   Terminal 2:
   cd project
   npm run dev

   → Change PORT in .env to 3002
   → Update VITE_API_URL to http://localhost:3002
